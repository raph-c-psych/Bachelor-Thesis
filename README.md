#  Investigating the Influence of Speed–Accuracy Integration on the Psychometric Properties of Cognitive Control

## Project Background

- **Type of Work:** Bachelor Thesis
- **Institution:** Institute of Psychology, RWTH Aachen University 
- **Author:** Raphael Christmann
- **Supervisor:** Dr. Jean-Paul Snijder-Steinhilber

## Instruction for Use with intended data
1. Get Data from Snijder et al. (2022) repository (https://osf.io/pqvga/overview)
2. Create a folder in parent folder of this repository for data named 'bachelor_data'
3. In bachelor_data create three folders 'data_raw', 'data_study_1' and 'data_study_2'
4. Put data from repository in 'data_raw'
5. Set this repository to working directory
